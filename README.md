# Learning Play Kit

태블릿에서 사용할 수 있는 로컬/정적 웹 기반 학습 게임 프로젝트입니다.

현재 첫 번째 학습 콘텐츠는 한국어문회 공인한자능력시험 배정한자 기준의 `한자 학습팩`입니다. 이후 영어 단어, 수학 연산, 과학 용어, 역사 인물, 국어 어휘처럼 다양한 학습 데이터를 같은 게임 구조에 얹을 수 있도록 확장하는 것을 목표로 합니다.

## 현재 포함된 콘텐츠

### 한자 학습팩

- 8급, 7급II, 7급 선택
- 한자 카드와 뜻·음 카드 매칭
- 뜻 고르기, 한자 고르기, OX 퀴즈
- 틀린 단어를 다시 풀 수 있는 복습함 모드
- 카드 수 선택
- 오늘의 미션, 보석, 스티커 보상
- 힌트, 섞기
- 태블릿 대응 반응형 UI

## 확장 방향

이 프로젝트는 특정 과목 하나에만 묶이지 않도록 아래 단위로 확장할 수 있습니다.

- `콘텐츠팩`: 한자, 영어 단어, 수학, 과학, 역사 등 학습 데이터
- `게임 모드`: 메모리, 뜻 고르기, OX 퀴즈, 타임어택, 복습함
- `보상 시스템`: 보석, 스티커, 연속 학습, 마스터리 레벨
- `테마`: 초등 저학년, 고학년, 심플 학습 모드 등 UI 스킨

## 실행

PC에서는 `index.html`을 브라우저로 열면 됩니다.

GitHub Pages에 올릴 경우 저장소 루트에 있는 `index.html`이 바로 실행됩니다.

## 파일 구조

```text
.
├── index.html
├── assets/
│   ├── learning-logo.png
│   ├── mode-*.png
│   ├── pack-*.png
│   └── reward-*.png
├── docs/
│   └── project-structure.md
├── .nojekyll
└── README.md
```

## 배포 메모

GitHub Pages를 사용할 때는 `assets/...`처럼 상대 경로를 유지해야 합니다. 현재 이미지는 스프라이트 좌표 방식이 아니라 모드와 보상별 개별 PNG 파일로 불러옵니다.
