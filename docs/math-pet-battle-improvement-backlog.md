# 수학펫 배틀 개선 필요사항 정리

작성일: 2026-06-29

## 검토 기준

- 기준 화면은 데스크톱 전투 화면이다.
- 모바일 화면은 현재 검토 범위에서 제외한다.
- 시안과 다른 화면 요소는 CSS로 임시 모사하지 않고, 가능한 경우 이미지 에셋을 생성하거나 기존 에셋을 보강해서 적용한다.
- 실제 플레이 가능한 기능 흐름을 유지하면서 시안 일치도를 높인다.

## 현재 반영된 개선 상태

- 전투 화면은 상단 HUD, 중앙 전투장, 우측 펫/팀/아이템 패널, 하단 문제 선택 패널 구조로 재구성됐다.
- 기본 전투 상태에서는 안내 문구, 콤보, 로그를 숨기고 수식과 선택지를 중심으로 표시한다.
- 정답 선택 후에는 전투 이펙트, 안내 문구, 다음 문제 버튼이 표시된다.
- 이름표, 커맨드 패널, 선택 버튼, 펫 카드, 스킬칩, 팀 슬롯, 아이템 슬롯은 이미지 에셋 기반으로 교체됐다.
- 데스크톱 기준 전투 진입, 정답 선택, 피드백 표시, 다음 문제 이동 흐름은 동작한다.

## 1차 개선 필요사항

### 1. 전투 캐릭터 비율과 위치 보정

현재 구현은 시안 대비 조련사, 펫, 적의 크기와 전장 내 거리감이 다르다.

- 시안은 조련사와 펫이 더 크게 보이고, 펫이 전투의 주체로 더 명확하다.
- 현재 화면은 배경이 많이 보이는 대신 캐릭터가 상대적으로 작아 보인다.
- 펫 공격 주체 컨셉을 강화하기 위해 펫과 적의 시각적 비중을 키워야 한다.

필요 작업:
- 펫/적 렌더 크기 조정
- 조련사는 보조 역할로 유지하되, 펫보다 시각 우선순위가 낮게 보이도록 위치 조정
- 공격/피격 모션 시 이동 방향과 거리 재조정

필요 에셋:
- 펫 공격 자세 또는 공격 오버레이
- 적 피격 자세 또는 피격 오버레이
- 조련사 대기 포즈 보강이 필요하면 별도 생성

### 2. 펫 상세 패널의 시안 일치도 향상

현재 우측 활성 펫 패널은 기능적으로 구성됐지만, 시안의 카드 구조와 비교하면 정보 밀도와 시각 구조가 단순하다.

- 시안은 속성 아이콘, 펫 이미지, 스킬 버튼이 명확히 분리되어 있다.
- 현재는 펫 이름/속성/스킬 설명이 한 카드 안에 압축되어 보인다.
- 스킬명과 설명을 분리해 읽기 쉽게 만들어야 한다.

필요 작업:
- 활성 펫 패널을 `속성 아이콘 + 펫 이미지 + 펫 이름 + 레벨 + 스킬 슬롯` 구조로 재정리
- 스킬 설명 대신 스킬명 중심으로 보여주고, 상세 설명은 툴팁 또는 하단 보조 문구로 분리
- 펫 속성별 컬러 또는 아이콘 적용

필요 에셋:
- 물별, 풀잎, 번개 등 속성 아이콘
- 스킬 슬롯 강조 프레임
- 펫 카드 내부 배지 프레임

### 3. 팀 슬롯 정보 강화

현재 팀 슬롯은 3개의 펫 상태를 보여주지만, 시안에 비해 슬롯 내 정보가 부족하다.

- 시안은 각 펫 슬롯에 HP 또는 상태 바가 포함되어 있다.
- 현재는 펫 이미지와 이름 중심이라 전투 팀 관리 느낌이 약하다.
- 잠긴 펫과 보유 펫의 차이는 보이지만, 전투 준비 상태가 충분히 드러나지 않는다.

필요 작업:
- 보유 펫 슬롯에 작은 HP 바 또는 레벨 표시 추가
- 활성 펫 슬롯의 선택 상태를 더 강하게 표시
- 잠긴 슬롯은 획득 조건을 알 수 있도록 개선

필요 에셋:
- 소형 HP 바 프레임
- 활성 슬롯 강조 프레임 보강
- 잠김 슬롯 아이콘 또는 어두운 실루엣 에셋

### 4. 아이템 슬롯의 사용 가능 상태 개선

현재 아이템 슬롯은 수량과 아이콘을 보여주지만, 사용 가능/불가 상태의 차이가 시안만큼 명확하지 않다.

- 사용 가능한 아이템, 수량 0 아이템, 장비 슬롯이 같은 계열로 보여 즉시 구분이 약하다.
- 전투 중 사용할 수 있는 아이템은 더 명확히 강조되어야 한다.
- 장비 상점에서 준비되는 장비형 아이템은 전투 아이템과 구분이 필요하다.

필요 작업:
- 사용 가능 아이템 슬롯 강조
- 수량 0 아이템은 어둡게 처리하되 텍스트 가독성 유지
- 장비 아이템은 별도 장비 슬롯 또는 장착 상태로 분리

필요 에셋:
- 사용 가능 슬롯 프레임
- 비활성 슬롯 프레임
- 장비 장착 슬롯 프레임

### 5. 하단 문제 패널의 시안 정밀도 향상

현재 하단 패널은 시안의 큰 구조를 따르지만 세부 비율은 아직 차이가 있다.

- 시안은 문제 영역과 선택지 사이의 여백이 더 균형적이다.
- 현재는 문제 숫자가 큰 편이고, 선택지 버튼이 하단에 더 압축되어 보인다.
- 선택지 버튼은 시안보다 납작하고 좌우 여백이 좁다.

필요 작업:
- 문제 수식 높이와 선택지 영역 높이 비율 재조정
- 선택지 버튼 크기와 내부 숫자 위치 보정
- 정답/오답 상태에서 버튼 색과 테두리 에셋이 기본 버튼과 동일한 픽셀 품질을 유지하는지 재검토

필요 에셋:
- 선택지 버튼의 정답/오답/비활성 버전
- 문제 패널 내부 장식 프레임 보강

### 6. 전투 피드백 연출 강화

현재 정답/오답 피드백은 이미지와 안내 문구로 표시되지만, 펫이 실제로 공격을 주고받는 느낌은 아직 약하다.

- 정답 시 펫 공격 방향, 적 피격 반응, 데미지 숫자가 더 명확해야 한다.
- 오답 시 적 반격 또는 보호막 발동이 더 분명해야 한다.
- 펫 특성 발동 시 일반 정답과 다른 연출이 필요하다.

필요 작업:
- 정답: 펫 공격 이펙트 -> 적 피격 -> 데미지 안내 순서 강화
- 오답: 적 반격 또는 방어 이펙트 -> 펫 HP 변화 안내 강화
- 특성 발동: 두 번 공격, 보호막, 회복 등 별도 피드백 문구와 이펙트 적용

필요 에셋:
- 펫 공격 이펙트 단일 이미지
- 적 피격 이펙트 단일 이미지
- 두 번 공격 전용 이펙트
- 보호막/회복/속도 특성 전용 이펙트
- 데미지 숫자 팝업 프레임

## 2차 개선 필요사항

### 1. 월드맵 지역 배치 재검토

전투 화면은 개선됐지만, 월드맵 지역 노드는 배경과 완전히 맞물린 느낌이 아직 부족하다.

필요 작업:
- 지역 노드가 배경의 길, 건물, 광장, 상점 위치와 자연스럽게 맞도록 재배치
- 전투 숲길, 펫 목장, 장비 상점, 아이템샵, 보상 상자의 이동 흐름 정리
- 각 지역 진입 전 상태 안내 또는 잠금 조건 표시

필요 에셋:
- 지역별 랜드마크 단일 에셋 보강
- 월드맵 노드 선택 강조 이펙트

### 2. 상점/아이템샵/목장 화면의 시안 일관성 보강

현재 전투 화면의 시안 일치도가 올라간 만큼, 다른 화면도 같은 픽셀 UI 시스템으로 맞춰야 한다.

필요 작업:
- 장비 상점, 아이템샵, 펫 목장 화면의 패널 프레임 통일
- 구매 버튼, 장착 버튼, 펫 선택 버튼을 이미지 에셋 기반으로 교체
- 지역별 배경과 카드 구조를 전투 화면과 같은 품질로 정리

필요 에셋:
- 상점 상품 카드 프레임
- 구매 버튼 프레임
- 장착 상태 배지
- 펫 목장 슬롯 프레임

### 3. 한글 텍스트 깨짐 정리

일부 기존 문서와 코드 내 한글 문자열이 깨져 보이는 구간이 있다.

필요 작업:
- 실제 화면에 노출되는 문자열 우선 점검
- 문서 파일 인코딩 점검
- 코드 내 깨진 문자열은 한글 기준으로 복구
- 저장/배포 환경에서 UTF-8 유지 확인

### 4. 저장 데이터와 진행 상태 표시 개선

현재 진행 데이터는 로컬 스토리지 기반으로 유지된다. 사용자가 저장 상태를 이해하기 쉽도록 화면 내 표시가 필요하다.

필요 작업:
- 진행 저장 상태 안내 문구 또는 아이콘 추가
- 스테이지 클리어, 보상 획득, 상점 구매 기록이 명확히 반영되는지 점검
- 초기화 또는 새로 시작 기능 검토

## 우선순위 제안

1. 전투 캐릭터 비율과 위치 보정
2. 하단 문제 패널 정밀도 향상
3. 전투 피드백 연출 강화
4. 우측 펫/팀/아이템 패널 정보 구조 개선
5. 월드맵 지역 배치 재검토
6. 상점/목장 화면 픽셀 UI 통일
7. 한글 텍스트 깨짐 정리
8. 저장 데이터 상태 표시 개선
