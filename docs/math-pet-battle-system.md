# 수학 펫 배틀 시스템 기획

## 기본 방향

초등학교 저학년 대상의 덧셈/뺄셈 학습 게임이다. 플레이어 캐릭터는 직접 공격하지 않는 `조련사`이며, 문제의 정답을 선택하면 펫이 행동한다. 모든 화면 문구, 펫 이름, 아이템 이름, 특성 설명은 한글을 기본 언어로 사용한다.

## 핵심 플레이 루프

1. 스테이지에 입장한다.
2. 적 펫 또는 장애물이 등장한다.
3. 덧셈/뺄셈 문제가 제시된다.
4. 정답을 선택하면 아군 펫이 공격하거나 특성을 발동한다.
5. 오답을 선택하면 적이 반격하거나 방어형 특성이 발동한다.
6. 전투 종료 후 별코인, 정답보석, 경험치를 얻는다.
7. 펫 성장, 장비 장착, 아이템 구매로 다음 스테이지를 준비한다.

## 전투 규칙

- 문제 유형: 한 자리/두 자리 덧셈, 한 자리/두 자리 뺄셈
- 선택지 수: 3개
- 기본 정답 보상: 아군 펫 기본 공격 1회
- 기본 오답 페널티: 적 펫 반격 1회
- 연속 정답: 3연속부터 추가 보상 확률 증가
- 전투 패배: 스테이지 재도전, 획득 보상 일부 유지
- 조련사 역할: 펫 선택, 정답 선택, 아이템 사용, 스킬 발동 보조

## 펫 능력 구조

각 펫은 `속성`, `기본 공격`, `특성`, `성장 방향`을 가진다.

| 펫 | 속성 | 특성 | 효과 | 성장 방향 |
| --- | --- | --- | --- | --- |
| 모모 | 물별 | 두번톡톡 | 정답 시 25% 확률로 2회 공격 | 공격형 |
| 루루 | 잎방울 | 보호잎 | 오답 시 1회 보호막 생성 | 방어형 |
| 토리 | 번개씨 | 빠른발 | 2연속 정답 시 선제 공격 | 속공형 |
| 포포 | 구름솜 | 포근회복 | 정답 시 아군 HP 소량 회복 | 회복형 |
| 비비 | 반짝 | 보석눈 | 정답보석 획득량 증가 | 보상형 |
| 코코 | 흙콩 | 단단껍질 | 받는 피해 감소 | 안정형 |

## 적 구성

적은 초반에는 위협적이기보다 친근한 형태로 구성한다. 적 이름도 학습/숫자 테마를 유지한다.

| 적 | 특징 | 사용 스테이지 |
| --- | --- | --- |
| 숫자젤리 | 가장 기본적인 적, 낮은 체력 | 1장 숲길 |
| 덧셈도토리 | 정답 시 추가 코인 드롭 | 1장 숲길 |
| 뺄셈게 | 오답 시 반격이 강함 | 2장 해변 |
| 시계버섯 | 제한 시간 문제에 등장 | 2장 해변 |
| 상자미믹 | 보상 상자 앞에서 등장 | 보너스 스테이지 |

## 아이템 구성

| 아이템 | 유형 | 효과 |
| --- | --- | --- |
| 별목걸이 | 장비 | 공격형 특성 발동 확률 증가 |
| 방패배지 | 장비 | 오답 피해 감소 |
| 집중쿠키 | 소비 | 다음 문제 선택지 하나 제거 |
| 힌트나침반 | 소비 | 정답 후보를 은은하게 강조 |
| 반짝간식 | 성장 | 펫 경험치 증가 |
| 회복젤리 | 소비 | 아군 펫 HP 회복 |
| 행운주머니 | 장비 | 별코인 획득량 증가 |
| 연습연필 | 소비 | 오답 1회를 무효 처리 |

## 재화와 보상

| 재화 | 용도 | 획득처 |
| --- | --- | --- |
| 별코인 | 상점 구매, 장비 구매 | 스테이지 클리어, 상자 |
| 정답보석 | 펫 해금, 특성 강화 | 연속 정답, 보너스 문제 |
| 성장별 | 펫 레벨업 | 전투 승리, 일일 미션 |

## 화면 구성 우선순위

1. 메인 전투 화면
2. 펫 상세/성장 화면
3. 상점 화면
4. 스테이지 선택 화면
5. 보상 결과 화면
6. 펫 도감 화면
